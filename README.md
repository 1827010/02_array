# 今回の問題

配列の勉強です。
入力される文字を配列に順次格納して、取り出すように指示されたら、その値を出力してください。

## 入力される値
入力は以下のフォーマットで与えられます。
~~~
N
n
n
n
n
~~~
* N: 入力されるデータの数。
* n: 正の値なら、配列に格納、負の値なら、その絶対値の順番に格納された数字を出力

## 期待する出力

入力されたデータの負の回数だけ数字を出力します。

最後は改行し、余計な文字、空行を含んではいけません。

## 条件
すべてのテストケースで以下の条件を満たします。
入力される値は全て整数

* 2 ≦ N ≦ 100
* -99 ≦ N ≦ 100

## 入力例1
~~~
2
3
-1
~~~

## 出力例1
~~~
1
~~~

## 入力例2
~~~
2
5
7
8
-2
4
-4
~~~

## 出力例1
~~~
7
4
~~~
